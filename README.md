# Amazon_sales_report_dashboard
📊 Amazon Sales Dashboard
# 🔹 Project Description

This project contains an Amazon Sales Dashboard built in Power BI, powered by raw sales data. The dashboard provides an interactive and visual representation of key performance indicators (KPIs) to help understand sales patterns, revenue distribution, and customer behavior.

The dataset (Amazon sheet.csv) was cleaned and modeled in Power BI to generate meaningful insights.

# 🔹 Features

.Sales performance tracking across time periods

.Revenue distribution by product, category, region, and payment method

.Identification of top-selling products and high-revenue regions

.Profit analysis and trend visualization

.Dynamic filters & slicers for interactive exploration

# 🔹 Tools & Technologies

.Power BI – for building the dashboard

.CSV dataset – raw sales data source

.Data modeling – calculated columns & DAX measures

# 🔹 Insights & Analysis

.From the dashboard, we can analyze:

.Top-selling product categories that generate the highest revenue.

.Regional sales performance, highlighting which regions contribute most to total sales.

.Payment method trends, showing customer preferences.

.Profit margins across different products.

.Sales over time, allowing seasonal trend identification.

.This dashboard can help e-commerce managers or analysts make data-driven decisions to optimize sales strategy.

# 🔹 How to Use

.Open the .pbix file in Power BI Desktop.

.Load the dataset (Amazon sheet.csv) if needed.

.Explore the interactive dashboard with filters and visual
